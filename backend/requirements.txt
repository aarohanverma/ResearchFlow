# ─── PDF Parsing ──────────────────────────────────────────────────────────────
# Default parser: Marker (PDF_PARSER=marker). Runtime fallback chain:
#   <chosen> → Marker (always in chain) → Gemini Vision → minimal placeholder
#
# Docling is opt-in (PDF_PARSER=docling). It provides richer section hierarchy,
# tables, equations, and figures but pulls in PyTorch + EasyOCR which can spike
# RAM by ~2 GB on first parse — unsuitable for small VMs without explicit opt-in.
# Docling is NOT auto-added to the fallback chain to prevent silent OOM crashes.
#
# On CPU-only hosts install the slimmer CPU torch wheel before pip-installing:
#   pip install docling --extra-index-url https://download.pytorch.org/whl/cpu
